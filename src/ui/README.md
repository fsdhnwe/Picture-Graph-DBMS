# 多媒體圖像搜尋系統 UI

這是一個使用 Gradio 構建的圖形介面，用於多媒體圖像資料庫的管理和搜尋。

## 功能

- **上傳圖片**：將圖片上傳到 Neo4j 圖資料庫中
- **搜尋圖片**：基於文本描述搜尋相關圖片，支持語義搜尋
- **圖片問答**：基於圖片內容回答問題

## 安裝要求

確保已安裝以下依賴項：

```
gradio>=4.0.0
langchain>=0.1.0
torch>=2.0.0
torchvision>=0.15.0
transformers>=4.30.0
pillow>=10.0.0
```

## 使用方法

1. 確保 Neo4j 資料庫已經設置好並運行中
2. 運行啟動腳本：

```bash
python run_app.py
```

或者帶參數運行：

```bash
python run_app.py --share --debug
```

參數說明：
- `--share`：生成一個可公開訪問的鏈接
- `--debug`：啟用除錯模式

## 界面說明

### 上傳圖片頁面
- 選擇或拖放圖片
- 可選填寫圖片描述
- 點擊上傳按鈕將圖片添加到資料庫

### 搜尋圖片頁面
- 輸入搜尋關鍵詞
- 調整相似度閾值（值越高要求越嚴格）
- 點擊搜尋按鈕顯示結果
- 點擊圖片可查看詳情

### 圖片問答頁面
- 輸入與圖片相關的問題
- 系統將搜尋相關圖片並嘗試回答

## 注意事項

- 上傳的圖片將保存在項目的 `test_images` 目錄中
- 請確保 Neo4j 資料庫配置正確，可在 `src/config.py` 中設置連接參數
